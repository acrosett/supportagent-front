Easy Integration

Get up and running in minutes with our simple 2-step integration

1

Embed the Chat Widget

Add our chat widget to your website with a unique token

2

Setup Webhook

Create a webhook endpoint to validate user tokens

Step 1: Embed the chat widget

<script>
  (function() {
    var widget = document.createElement('script');
    widget.src = 'https://cdn.aisupportagent.com/widget.js';
    widget.setAttribute('data-token', 'YOUR_UNIQUE_TOKEN');
    document.head.appendChild(widget);
  })();
</script>

Control Widget Functions

// Pass a callback to get current user token
window.AISupportWidget.setUserTokenCallback(function getUserToken() {
  // Return the current logged user token
  return localStorage.getItem('userToken'); // or your auth method
});

// Show the widget
window.AISupportWidget.show();

// Hide the widget
window.AISupportWidget.hide();

Step 2: Webhook endpoint (on your server)

GET /your_webhook/validate-token?userToken=USER_TOKEN&shared_secret=YOUR_SHARED_SECRET

Response:
{
  "valid": true,
  "userInfo": {
    "name": "John Doe",
    "email": "john@example.com",
    "unique_id": "USER_UNIQUE_ID",
    "support_level": "vip"
  }
}
